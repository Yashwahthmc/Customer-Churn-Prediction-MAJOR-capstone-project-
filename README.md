# Customer Churn Prediction

## 📌 Overview

This project predicts whether a customer is likely to churn using machine learning techniques. It helps businesses identify at-risk customers and take proactive retention measures.

---

## 📊 Dataset

* Telco Customer Churn Dataset
* Contains 7043 customer records and 21 features

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* XGBoost
* Matplotlib, Seaborn
* SHAP (Explainable AI)

---

## 🔍 Project Workflow

1. Data Preprocessing
2. Feature Engineering
3. Model Training
4. Model Evaluation
5. Visualization
6. Explainability using SHAP

---

## 🤖 Models Used

* Logistic Regression
* Random Forest
* XGBoost

---

## 📈 Results

* Best Model: Logistic Regression
* Accuracy: ~79.8%
* ROC-AUC Score: ~0.84

---

## 📊 Key Insights

* Customers with low tenure are more likely to churn
* Higher monthly charges increase churn probability
* Fiber optic users show higher churn rates
* Long-term contracts reduce churn

---

## 🧠 Explainability

SHAP was used to interpret model predictions and understand feature importance.
